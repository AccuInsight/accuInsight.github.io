FAQ

Q. AccuInsight+란 무엇입니까? AccuInsight+는 Accuracy + Insight 합성어로 데이터 분석에 있어 정확도를 높이고 통찰을 키운다는 의미를 갖고 있습니다. AccuInsight+는 Data Science를 지원하는 머신 러닝 플랫폼 서비스이며 Amazon 분석서비스 (AWS) 처럼 다양한 서비스를 구독형 / 설치형으로 지원합니다. 분석 Sandbox 생성 / 데이터 처리 / 모델 학습 및 배포 등 고객의 End-to-End 분석활동의 모든 활동을 지원하며, 다양한 사용자에게 다양한 workspace / workflow를 제공하여 분석 프로젝트의 성공을 돋는 서비스로 구성되어 있습니다.

Q. AccuInsight+는 어떤 제품으로 구성되어 있습니까? Pipeline : Data EDA / Pre-processing 등을 포함한 ML Pipeline 작성을 Drag & Drop으로 제공해주는 서비스. Modeler : GPU/CPU기반의 분석 Workspace를 제공하며, R Studio / Jupyter Notebook등을 사용자/그룹별로 제공할 수 있음 이 외에도 분석 실행을 위한 Sandbox형 하둡 클러스터를 제공하는 DHP ( Dynamic Hadoop Provisioning )과 AWS와 GCP의 클러스터를 연동하는 MCM ( Multi Cloud Management )가 옵션으로 제공됩니다.

Q. AccuInsight+에 대한 가격 정책은 어떻게 됩니까? Public Service는 사용하시는 만큼 내는 과금제로 사용 가능합니다. 설치형은 별도의 라이선스 계약을 기준으로 제공하고 있습니다. 자세한 라이선스 금액은 영업 채널을 통해 확인 바랍니다.

Q. 다른 데이터 분석 솔루션과 기본적으로 어떤 차이를 가지고 있습니까? 가장 큰 특징으로는 Data Science를 활용한 분석 프로젝트를 위한 End-to-End서비스를 제공하고 있습니다. 분석 실행을 위한 인프라 생성 데이터 전처리 및 워크플로우를 Bigdata 분산처리 기반으로 제공 모델 개발 및 협업툴을 제공하고 API Serving까지 제공 을 모두 제공하는 차이점을 가지고 있습니다.

Q. AccuInsight+를 도입하기 위해서는 어떤 인프라가 필요합니까? 설치형으로 사용할 때에는 k8s cluster가 기본적으로 필요합니다. 기본적으로 보유하고 계신 k8s 클러스터가 있다면 그곳에 설치가 가능하고, 그밖에 퍼블릭 클라우드 환경의 AWS EKS / GCP GKE와 같은 Managed Service에서도 설치가 가능합니다.

Q. 사내 시스템으로 도입을 위해서는 일정부분 Customizing이 필요할 수 있습니다. 지원이 가능한가요?솔루션 라이선스와 별도로 Customizing비용 산정이 별도로 필요합니다. 또한 분석 과제 실행을 위한 Engineering or Scientist 지원 등을 Expert Service로 제공하고 있습니다.

Q. Q. 사용자 교육이 필요한가요? 필요하다면 기본으로 제공 하는 것인지?네, 기본적으로 사용자 매뉴얼 등의 Document를 제공하며 사용성을 높이기 위한 Hands-on 교육을 제공하고 있습니다.

Q. 사용에 필요한 기본적인 지식이 있을까요?분석에 큰 지식이 없는 일반 사용자도 Pipeline을 통해 직관적인 UI 및 Drag & Drop을 통해 Bigdata 분산처리 기반의 빠른 분석 작업을 실행할 수 있습니다. 고급 분석 사용자는 Modeler를 통해 본인의 독립된 workspace에서 작업이 가능하며, 작성한 Code와 Model을 다른 사용자에게 공유할 수 있습니다. 또한 생성된 모델을 바로 운영에 배포가 플랫폼 내부에서 제공되므로 운영시스템 전환 시 개발자의 도움이 없이 가능합니다.

Q. Pipeline 구동을 위해서는 hadoop cluster가 필요합니까?네. AccuInsight+ 서비스인 DHP를 사용하여 Sandbox 클러스터를 생성하여 활용할 수도 있으며, 내부에 별도로 운영중인 On-prem. cluster가 있다면 Pipeline과 연동이 가능합니다.

Q. Pipeline에서는 어떤 알고리즘이 지원 가능합니까?데이터 샘플링 (random, systematic, stratified, cluster), 데이터 처리 (filter, sort, replace, join, distinct, union, pivot, dropNa, nearZeroVal 등) 피처 엔지니어링 (imputer, bucketizer, normalizer, chiSquare/corr/tree selector, robust/standard/minMax/maxAbs scaler, one hot encoder, PCA 등) ML 학습 (ATFSurvive, DecisionTree, GBT, Linear, Isotonice, RF, XGB, ALS, KMeans, GaussianMixture, LDA, NaiveBayes, Logistic 등) 을 제공합니다. 자세한 기능 요건은 매뉴얼을 통해 확인할 수 있습니다.
