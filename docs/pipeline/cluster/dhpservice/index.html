DHP 서비스#

사용자가 접근할 수 있는 클러스터 중 DHP를 통해 생성한 클러스터만 조회할 수 있습니다. [클러스터] > [DHP 서비스] 로 이동합니다.

  1. 클러스터검색 : 조회대상 목록이 많은 경우 클러스터명을 직접 입력하여 검색
  2. 날짜검색 : 클러스터 생성일 기준으로 검색(오늘 날짜 기준으로 6개월간 생성된 클러스터 조회)
  3. 클러스터목록 : 사용자가 접근할 수 있는 클러스터의 상세정보(ID, 상태, 이름, 생성시간 등) 조회

날짜검색항목의 캘린더 아이콘을 선택하여 당일, 1주, 1개월, 3개월, 6개월내 생성된 클러스터만 검색할 수 있습니다. 기본값은 오늘날짜를 기준으로 6개월이며, 6개월이 지난 클러스터 검색시 직접입력할 수 있습니다.
