변수관리 개요#

워크플로우 실행 시 사용하는 글로벌 변수를 관리하는 메뉴입니다. 생성한 변수 그룹은 워크플로우 생성 화면의 parameter 탭 globalVariable에서 선택하여 사용할 수 있으며 parameters에 선언된 변수보다 우선순위가 높습니다.

  1. 검색 : 검색할 변수를 입력하여 검색
  2. 추가 : 변수를 추가
  3. 삭제 : 삭제할 변수를 목록에서 체크 후 삭제 버튼을 클릭하여 삭제
  4. 목록
    • 그룹 명 : 변수를 구분하기 위한 이름
    • 변수 목록 : 변수 그룹에서 사용하는 변수 목록
    • 변수 개수 : 변수 그룹에서 사용하는 변수 개수
    • 반복 횟수 : 변수의 값을 다르게 하여 워크플로우 잡을 수행할 횟수
  5. 편집 : 클릭하여 변수 정보 편집
  6. 삭제 : 클릭하여 해당 변수 삭제

변수 편집기#

변수 정보를 생성/수정합니다.

  1. 그룹 명 : 변수 정보를 구분하기 위한 변수 그룹 명
  2. 변수 추가 : 사용할 변수를 추가
  3. 반복 추가 : 반복 횟수를 추가 하며 변수의 값을 다르게 설정하여 잡을 수행
  4. csv 불러오기 : csv 파일로 작성된 변수 정보를 로드 할 수 있으며, 추가되는 변수 값의 행 수만큼 반복 횟수가 추가되고 use_yn은 Y로 셋팅
  5. csv 내보내기 : 변수 정보를(변수 명, 변수 값) csv 파일로 다운로드
  6. 저장 : 변수 정보를 저장
  7. 반복 : 반복 횟수를 추가하는 경우 행이 추가 되며, 반복 횟수만큼 잡에서 수행
  8. use_yn : 반복 수행하는 경우 체크
  9. 변수 : 변수를 추가하는 경우 추가하는 변수 명으로 컬럼이 추가 되며 워크플로우에서 필요한 변수 개수만큼 추가하여 사용가능
