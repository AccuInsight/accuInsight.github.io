addTimestamp

Definition#

timestamp 컬럼을 추가한다. 좌측 [데이터처리(고급)]노드 중 [addTimestamp]노드를 drag & drop 한 후 Property 항목을 입력합니다.

Set#

[setting], [parameter] 설정은 [워크플로우 생성] > [설정]을 참고합니다.

property#

[Node Description] 작성 중인 노드명 입력

  1. newColumn : 추가 컬럼명 입력
  2. timestampValue : 날짜값 입력(ex. now-1d , now-1h)
  3. timestampFormat : 날짜 형식 입력(ex. yyyyMMddHHmmss)

Example#

상품 주문정보에 하루 전 일자 정보를 "yyyy-MM-dd" 타입 형태로 저장하는 컬럼을 추가하는 워크플로우를 생성합니다.

  1. [HDFS불러오기], [addTimestamp] 노드를 Designer에 Drag & Drop하여 워크플로우 생성
  2. [addTimestamp]노드에 아래와 같이 입력 후 snapshot 클릭
