addConditionCol

Definition#

조건식을 입력하여 True/False에 매핑되는 값을 새로운 컬럼에 추가합니다.
좌측 [데이터처리(고급)]에서 [addConditionCol]노드를 drag & drop 한 후 Property 항목을 입력합니다.

Set#

[setting], [parameter] 설정은 [워크플로우] > [생성] > [기본구성]을 참고합니다.

property#

[Node Description] 작성 중인 노드명 입력

  1. expression : 사용자가 원하는 조건식 직접 입력(띄어쓰기 필수). 적용할 조건식이 여러 개 일 경우 오른쪽 [+]버튼 선택
    • customCondition : 조건식
    • trueValue : 조건식이 True일 경우 매핑 될 값
    • falseValue : 조건식이 False일 경우 매핑 될 값
    • newColumn : expression 결과가 입력될 새로운 컬럼(미입력시 조건식을 적절하게 지환[필드명조건값]하여 지정)

Example#

상품 주문정보에서 dates 컬럼 데이터가 주말일 경우 holiday 테이블에 매핑값('1')을 입력하는 워크플로우를 생성합니다.

  1. [HDFS불러오기], [convertDate], [addCoditionCl] 노드를 Designer에 Drag & Drop하여 워크플로우 생성
  2. [ConvertDate]노드를 활용하여 dates 컬럼 자료에서 요일정보 추출 후 day컬럼에 저장([ConvertDate]노드 상세설명은 해당 매뉴얼 참고)
