removeTaAuto

Definition#

Trun Around를 기간 단위 기준으로 판단하여 해당 데이터 행을 삭제합니다. 예를 들어 특정 일자의 컬럼들 중 70% 이상이 0인것이 존재하면 해당 일자의 Row들은 전부 삭제합니다. 좌측 [데이터처리(고급)]노드 중 [removeTaAuto]노드를 drag & drop 한 후 Property 항목을 입력합니다.

Set#

[setting], [parameter] 설정은 [워크플로우 생성] > [설정]을 참고합니다.

property#

[Node Description] 작성 중인 노드명 입력

  1. col : 날짜 컬럼을 선택
  2. dateFormat : 날짜 포맷을 지정
  3. dateUnit : TA를 확인할 시간 단위
  4. ratio : TA의 비율

Example#

워크플로우를 구성합니다.

  1. [HDFS불러오기], [removeTaAuto] 노드를 Designer에 Drag & Drop하여 워크플로우 생성

  2. [removeTaAuto]노드에 아래와 같이 입력 후 snapshot 클릭
