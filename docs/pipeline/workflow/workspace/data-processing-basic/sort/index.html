sort

Definition#

데이터를 정렬합니다.
좌측 [데이터처리(기본)]노드 중 [sort]노드를 drag & drop 한 후 Property 항목을 입력합니다.

Set#

[setting], [parameter] 설정은 [워크플로우] > [생성] > [기본구성]을 참고합니다.

Property#

[Node Description] 작성 중인 노드명 입력

  1. columns : 정렬할 column을 선택하고 오름차순/내림차순 선택
    • 대상 컬럼 : 목록 중 정렬 적용할 컬럼 선택
    • 정렬 기준 : asc(오름차순), desc(내림차순)

Example#

상품정보에서 category(asc), price(desc) 컬럼대상으로 정렬하는 워크플로우를 작성합니다.

  1. [HDFS불러오기], [sort] 노드를 Designer에 Drag & Drop하여 워크플로우 생성
  2. [sort] 노드를 아래와 같이 설정하고 snapshot 클릭
