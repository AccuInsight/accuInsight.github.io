unionAll

Definition#

두 개의 데이터 노드를 행 단위로 결합합니다.
좌측 [데이터처리(기본)]노드 중 [unionAll]노드를 drag & drop 한 후 Property 항목을 입력합니다. [unionAll]노드는 별도의 Property입력 항목이 없습니다.

Set#

[setting], [parameter] 설정은 [워크플로우] > [생성] > [기본구성]을 참고합니다.

Property#

[Node Description] 작성 중인 노드명 입력

Example#

상품정보 테이블 2종의 데이터를 행단위로 추가하는 워크플로우를 생성합니다.

  1. [HDFS불러오기], [unionAll] 노드를 Designer에 Drag & Drop하여 워크플로우 생성
  2. [unionAll] 노드 snapshot 클릭
