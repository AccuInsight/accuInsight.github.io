isin

Definition#

특정 값 포함 여부를 확인하여 그 결과를 True/False로 반환합니다.
좌측 [데이터처리(기본)]노드 중 [isin]노드를 drag & drop 한 후 Property 항목을 입력합니다.

Set#

[setting], [parameter] 설정은 [워크플로우] > [생성] > [기본구성]을 참고합니다.

Property#

[Node Description] 작성 중인 노드명 입력

  1. col : 기준 컬럼 선택
  2. argument : 찾을 문자열 입력. 적용대상 문자열이 여러 개일 경우 우측 [+]버튼으로 추가 가능
  3. newColumn : True/False 결과가 입력될 새로운 컬럼명 입력

argument 재사용#

[isin]노드에서 입력하는 argument를 저장하여 재사용할 수 있습니다. 문자열 입력 후 저장버튼을 누르면 argument가 저장됩니다.

argument저장내역은 변수목록 클릭시 팝업되며, 해당 목록을 적용시 argement항목에 자동 입력됩니다.

category 컬럼에서 "악세사리", "패션소품" 키워드를 찾아 그 결과를 accessories라는 새로운 컬럼에 True/False로 저장합니다.
