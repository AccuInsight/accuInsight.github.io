cast

Definition#

데이터 타입을 다른 타입으로 변환하는 노드입니다.
좌측 [데이터처리(기본)]에서 [cast]노드를 drag & drop 한 후 Property 항목을 입력합니다.

Set#

[setting], [parameter] 설정은 [워크플로우] > [생성] > [기본구성]을 참고합니다.

Property#

[Node Description] 작성 중인 노드명 입력

  1. subset : 변경대상 컬럼 및 데이터 타입 선택. 대상 컬럼이 여러 개일 경우 좌측 [+]버튼 선택
    • col : 변경대상 컬럼
    • dataType : 변경하고자 하는 데이터 타입

Example#

상품정보 데이터 중 group컬럼을 substr하여 age컬럼을 새로만들고, 데이터 타입을 Integet로 변경합니다.

  1. [HDFS불러오기], [substr], [cast] 노드를 Designer에 Drag & Drop하여 워크플로우 생성
  2. [substr]노드에서 group컬럼의 1,2번째 데이터로 age컬럼 생성(group, age = string)
  3. [cast]노드에서 age컬럼의 데이터타입을 Interger로 변경(group = string, age = integer )
