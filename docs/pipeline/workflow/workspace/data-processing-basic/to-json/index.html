toJson

Definition#

결과값을 json 포맷으로 변환합니다.
좌측 [데이터처리(기본)]노드 중 [toJson]노드를 drag & drop합니다. [toJson]노드는 별도의 Property입력 항목이 없습니다.

Set#

[setting], [parameter] 설정은 [워크플로우] > [생성] > [기본구성]을 참고합니다.

Property#

[Node Description] 작성 중인 노드명 입력
