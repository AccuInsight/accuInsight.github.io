select

Definition#

필요 컬럼만 선택하여 조회합니다. 컬럼 선택 순서에 따라 노출 순서가 적용됩니다. 팝업기능을 제공하므로 노출순서를 확인할 수 있습니다.
좌측 [데이터처리(기본)]노드 중 [select]노드를 drag & drop 한 후 Property 항목을 입력합니다.

Set#

[setting], [parameter] 설정은 [워크플로우] > [생성] > [기본구성]을 참고합니다.

Property#

[Node Description] 작성 중인 노드명 입력

  1. col : 적용할 column 선택. 아래 2가지 방법 지원
    • 우측 수정버튼 활용하여 필요 컬럼 선택
    • 드롭다운 버튼에서 필요 컬럼 선택

Example#

상품정보에서 필요한 컬럼(goods id, category, price)만 선택하는 워크플로우를 작성합니다.

  1. [HDFS불러오기], [select] 노드를 Designer에 Drag & Drop하여 워크플로우 생성
  2. [select] 노드를 아래와 같이 설정하고 snapshot 클릭
