subStr

Definition#

문자열을 잘라서 새로운 컬럼으로 추가합니다.
좌측 [데이터처리(기본)]노드 중 [subStr]노드를 drag & drop 한 후 Property 항목을 입력합니다.

Set#

[setting], [parameter] 설정은 [워크플로우] > [생성] > [기본구성]을 참고합니다.

Property#

[Node Description] 작성 중인 노드명 입력

  1. subset : col, startPosition, length, newColumn 정보 입력. 적용대상 컬럼이 여러 개일 경우 우측 [+]버튼으로 추가
    • col : 기준 컬럼 선택
    • startPosition : 시작 위치 입력
    • length : 길이 입력
    • newColumn : 신규 컬럼 명 입력

Example#

상품정보에서 group컬럼을 숫자열과 문자열로 분리하여 새로운 컬럼들을 생성합니다.

  1. [HDFS불러오기], [select], [subStr] 노드를 Designer에 Drag & Drop하여 워크플로우 생성
