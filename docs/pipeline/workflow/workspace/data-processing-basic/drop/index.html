drop

Definition#

선택한 컬럼을 삭제하는 노드입니다.
좌측 [데이터처리(기본)]노드 중 [drop]노드를 drag & drop 한 후 Property 항목을 입력합니다.

Set#

[setting], [parameter] 설정은 [워크플로우] > [생성] > [기본구성]을 참고합니다.

Property#

[Node Description] 작성 중인 노드명 입력

  1. col : 삭제할 컬럼 선택

Example#

상품정보 데이터 중 group, style, timestamp컬럼을 제거합니다.

  1. [HDFS불러오기], [drop] 노드를 Designer에 Drag & Drop하여 워크플로우 생성
  2. [drop]노드 삭제대상 컬럼에 group, style, timestamp 선택
  3. [drop]노드 snapshot선택시 해당 컬럼 삭제후 데이터 프리뷰됨
