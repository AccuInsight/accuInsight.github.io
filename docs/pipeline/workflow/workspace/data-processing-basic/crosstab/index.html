crosstab

Definition#

열의 빈도 테이블을 계산하는 노드입니다.
좌측 [데이터처리(기본)]에서 [crosstab]노드를 drag & drop 한 후 Property 항목을 입력합니다.

Set#

[setting], [parameter] 설정은 [워크플로우] > [생성] > [기본구성]을 참고합니다.

Property#

[Node Description] 작성 중인 노드명 입력

  1. col1 : 빈도표 작성을 위한 첫 번째 기준컬럼 선택(결과테이블에서 행에 위치)
  2. col2 : 빈도표 작성을 위한 두 번째 기준컬럼 선택(결과테이블에서 열에 위치)

Example#

상품정보 데이터 중 category, group컬럼을 사용하여 빈도테이블을 계산합니다.

  1. [HDFS불러오기], [crosstab]노드를 Designer에 Drag & Drop하여 워크플로우 생성
  2. [crosstab]노드에서 col1=category, col2=group 선택시 category 기준으로 group 별 빈도테이블이 계산됨
