replace

Definition#

특정 컬럼의 값을 지정한 값으로 치환합니다.
좌측 [데이터처리(기본)]노드 중 [replace]노드를 drag & drop 한 후 Property 항목을 입력합니다. Property 패널의 [더보기+] 버튼을 누르면 입력가능한 전체 Property 항목을 볼 수 있습니다.

Set#

[setting], [parameter] 설정은 [워크플로우] > [생성] > [기본구성]을 참고합니다.

Property#

[Node Description] 작성 중인 노드명 입력

  1. to_replace : 변경전 값 입력(문자열 입력시 쌍따옴표("") 사용필요)
  2. value : 대치값(변경후 값) 입력(문자열 입력시 쌍따옴표("") 사용필요)
  3. subset : replace 적용 컬럼선택. 전체 컬럼 대상으로 할 경우 생략

Example#

상품정보 중 group컬럼의 값을 변경합니다("20대 초반" > "20대 초반 이하")
