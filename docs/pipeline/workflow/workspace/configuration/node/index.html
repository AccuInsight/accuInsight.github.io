Node

node#

Node는 워크플로우를 생성하기 위해 필요한 데이터탐색, 전처리 및 머신러닝 도구모음입니다. [데이터불러오기], [데이터샘플링][데이터처리하기(기본,고급)], [피처엔지니어링], [데이터내보내기], [Flow구성] 으로 그룹핑되어 있습니다.

노드영역별 주요기능은 다음과 같습니다.

  1. 데이터불러오기 : Sandbox 또는 Object Storage 등에 위치한 source data를 불러오기 위해 사용
  2. 데이터샘플링 : 분석대상 데이터 중 일부 샘플링
  3. 데이터처리하기 : 수집데이터의 이상치, 특이치, 오류 등을 제거하기 위한 처리함수와 코딩기반 정제작업을 위한 기능 제공(기본/고급)
  4. 피처엔지니어링 : 데이터 분석을 위해 데이터 테이블의 컬럼(특징)을 생성하거나 선택하기 위해 사용
  5. 데이터내보내기 : Pipeline 실행결과를 sandbox 또는 object storage 등에 저장하기 위해 사용
  6. 아래와 같은 Pipeline 구성을 위해 사용
    • 병렬방식으로 데이터 처리하거나, 선행작업 결과에 따라 후행작업을 결정해야 하는 경우
    • 이미 구축된 Pipeline을 불러와서 사용하는 경우(subworkflow)
    • 사용자 작성 code(java, spark python 등)를 실행하는 경우
    • 머신러닝 학습을 수행해야 하는 경우

워크플로우작성 중 다른 화면으로 이동할 경우 임시 저장 됩니다.
