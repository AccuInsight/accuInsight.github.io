management#

생성된 워크플로우의 목록을 조회하고, 바로 실행할 수 있습니다.

  1. 전체 워크플로우의 그룹, 상태, 워크플로우 명, 설명, 시간, 생성자 필드에 대한 키워드 검색

    • 노드 명으로 해당 노드 타입을 가지는 워크플로우를 검색가능
    • RDB, HIVE 노드의 경우 노드에 작성된 테이블명으로 워크플로우 검색가능

  2. 버튼을 클릭하여 목록에서 체크한 워크플로우에 대해 전체 적용

    버튼명설명
    전체 내보내기전체 워크플로우를 json 파일로 내보내기
    내보내기선택한 n 개의 워크플로우를 json 파일로 내보내기
    불러오기내보낸 워크플로우 json파일을 불러오기
    실행선택한 n개의 워크플로우를 실행
    정지선택한 n개의 워크플로우를 정지
    삭제선택한 n개의 워크플로우를 삭제

  3. 워크플로우 목록을 표시하며, 각 필드의 의미는 아래와 같음

    필드명설명
    아이디워크플로우 ID
    그룹워크플로우가 속한 그룹명
    워크플로우 명워크플로우 이름
    설명사용자가 작성한 Description
    생성된 시각워크플로우를 생성한 시간
    업데이트된 시각마지막으로 워크플로우를 수정한 시간
    생성자워크플로우를 생성한 사용자
    상태Active / Inactive
    상세정보‘상세정보’ 화면이동 버튼, ‘인스턴스 목록’ 화면이동 버튼, ‘편집’ 화면이동 버튼
    권한워크플로우의 권한 부여 팝업 버튼
    Action 버튼실행, 종료, 삭제(*각 인스턴스 상태에 따라 그 시점에 사용 가능한 메뉴가 활성화됨)

  4. [워크플로우상세정보] 해당 워크플로우의 상세정보 화면으로 이동

  5. [인스턴스목록] 해당 워크플로우에서 발생한 인스턴스 목록을 확인하는 화면으로 이동

  6. [편집] 해당 워크플로우의 편집화면으로 이동

  7. [권한] 해당 워크플로우에 대한 사용자권한 부여/편집화면 오픈. 워크플로우 조회/기록/실행 권한을 다른 사용자에게 부여가능

    • [없음] 워크플로우 부여 권한 제거. 권한이 제거된 사용자는 사용자검색란으로 이동(사용자삭제 아님)
    • [조회] 워크플로우 조회
    • [조회/실행] 워크플로우 실행/정지 가능
    • [조회/기록/실행] 워크플로우 편집 가능

  8. [워크플로우실행] 해당 워크플로우 실행

    • 실행 시 잡 파라미터와 잡이 실행될 클러스터를 선택가능

  9. [워크플로우정지] 해당 워크플로우 정지

  10. [워크플로우삭제] 해당 워크플로우 삭제

  11. 워크플로우를 실행한 이력이 있는 경우 가장 최근 실행한 인스턴스 상세 화면으로 이동
