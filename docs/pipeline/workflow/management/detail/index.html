detail#

워크플로우 목록에서 [워크플로우 상세정보]를 선택하면 사용자가 작성한 Workflow 상세정보를 확인할 수 있습니다.

흐름, 상세, 파라미터, 로그, JSON 뷰어 등 다섯개의 탭으로 구성되어 있으며 세부 사항은 아래와 같습니다.

흐름#

  1. ‘돌아가기’: 목록 화면으로 돌아가기 버튼
  2. 워크플로우 정보
    • 워크플로우 명: 사용자가 입력한 워크플로우 명
    • 그룹: 사용자가 입력한 해당 워크플로우가 소속되는 그룹명
    • 상태: 워크플로우의 현재 상태
  3. Action 버튼 (* 각 Instance의 상태에 따라 그 시점에 사용 가능한 메뉴 활성화)
    • 수정: 클릭하여 워크플로우 수정화면으로 이동
    • 실행: 클릭하여 워크플로우 실행
    • 정지: 클릭하여 워크플로우 정지
    • 삭제: 클릭하여 워크플로우 삭제
  4. 해당 워크플로우의 노드 항목들이 Workflow Designer 에서 구성 한대로 표시되며, 실행중인 경우 실시간으로 status 업데이트 (Status 상태에 따라 미리 정의된 색상으로 highlight 처리되어 표시)

상세#

  • 아이디 : 워크플로우 생성 시 할당되는 ID
  • 그룹: 해당 워크플로우가 속하는 그룹명
  • 작성자: 생성자 ID
  • 워크플로우명: 사용자가 입력한 워크플로우 이름
  • 설명: 사용자가 입력한 Description
  • 생성된 시각: 해당 워크플로우가 생성된 시각
  • 업데이트된 시각: 해당 워크플로우가 업데이트된 시각
  • Job Count: 실행 횟수

파라미터#

워크플로우 실행 시점의 Parameter Key-Value 정보를 표시합니다.

로그#

해당 워크플로우가 실행되는 동안 출력된 Log 정보를 표시합니다.

JSON 뷰어#

해당 워크플로우의 입력 정보를 JSON 뷰어로 확인할 수 있습니다.
