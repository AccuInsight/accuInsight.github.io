대시보드#

워크플로우 작업들의 Category 별 Running 상태, 작업 수행시간, 작업 실행 개수 등을 dashboard 메뉴에서 확인할 수 있습니다.

  1. 기간 : 조회하고자 하는 기간 선택
  2. 그룹별 상태 : Workflow의 Category 별 Running 및 그 외 상태 확인
  3. 그룹의 서비스 상세 현황 : [1.그룹별 상태]에서 선택한 Category의 Workflow들의 상세 작업들 상태 확인
  4. 실행시간 비교 : [1.그룹별 상태]에서 선택한 Category의 Category 별 작업 수행 시간을 확인 및 비교 가능
  5. 시간대별 워크플로우 실행 횟수 이력 : 전체 Workflow의 24시간 동안의 시간대 별 작업 실행 개수 현황 확인
  6. 예상 실행시간 대비 실제 실행시간 비교 : 선택한 Schedule Workflow의 실제 수행 시간, 수행 예상 시간, 수행 예상 시간 대비 초과 시간 확인
