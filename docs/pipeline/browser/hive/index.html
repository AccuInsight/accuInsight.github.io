HIVE 브라우저#

HIVE 브라우저 개요#

클러스터 hive에 접속해서 쿼리를 실행할 수 있습니다.

  1. 접근하고자 하는 Cluster 선택
  2. DB 선택
  3. 검색 : 검색할 테이블 명을 입력 후 ‘찾기’ 아이콘 클릭하여 검색
  4. 테이블 리스트 : 조회한 테이블 리스트로 특정 테이블에 샘플 데이터와 스키마 정보를 조회 가능
  5. 컬럼 : 테이블 리스트에서 선택한 table의 schema 리스트로 데이터 출력에서 제외할 column은 checkbox 해제 처리
  6. 쿼리 : 실행할 쿼리 입력
  7. Hive 계정관리 : Hive 접속 시 사용할 계정 정보 관리
  8. 새쿼리 : 쿼리 에디터 초기화
  9. 쿼리실행이력 : 실행한 쿼리의 이력 관리
  10. 쿼리저장목록 : 저장한 쿼리 관리
  11. 쿼리 실행 : 버튼을 클릭 또는 Ctrl+Enter로 쿼리 실행
  12. 저장 : 쿼리 에디터에서 작성중인 쿼리 저장
  13. 데이터 출력 : 4.테이블 리스트에서 선택한 table과 5.컬럼에서 선택한 column 기준의 데이터 또는 6. 쿼리에서 실행한 쿼리 결과의 column 기준의 데이터를 출력
  14. 데이터 검색 : 검색어를 입력한 뒤 ‘찾기’ 아이콘 클릭하여 데이터를 검색

HIVE 계정관리#

클러스터의 Hive 계정을 등록/수정/삭제 할 수 있습니다.

HIVE 쿼리실행이력#

쿼리 실행 이력을 관리하는 화면으로 쿼리의 실행 시각, 쿼리 내용, 상태, 수행 시간을 조회할 수 있으며 실행이력을 삭제 할 수 있습니다.

HIVE 쿼리저장이력#

저장한 쿼리를 관리하는 화면으로 조회, 검색, 삭제 가능합니다. 저장한 쿼리를 더블 클릭하면, 쿼리 화면으로 복사 되어 쿼리를 사용 할 수 있습니다.

[Note]
Hive에서 실행되는 쿼리라면 Hive 브라우저에서 실행할 수 있습니다.
숫자타입 데이터를 Insert into query로 입력할 경우, 싱글쿼테이션이 포함되면 null 값으로 표현됩니다.
