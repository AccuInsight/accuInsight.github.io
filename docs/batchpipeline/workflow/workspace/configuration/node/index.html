Node

#node 설정

Node는 크게 데이터 불러오기, 처리하기, 내보내기를 수행하는 ETL 노드와 그 외의 Action 노드 구성으로 나뉜다. ETL 노드는 작성 후에 source 버튼을 클릭하여 팝업에서 PySpark 코드로 변환된 것을 확인 할 수 있다.

워크플로우작성 중 다른 화면으로 이동할 경우 임시 저장 된다.
