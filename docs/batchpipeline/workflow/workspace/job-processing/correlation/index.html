correlation

Definition#

데이터의 correlation(상관분석)을 수행한다. 좌측 [Flow구성]]노드 중 [correlation]노드를 drag & drop 한 후 Property 항목을 입력한다. Property 패널의 [더보기+] 버튼을 누르면 입력가능한 전체 Property 항목을 볼 수 있다.

Set#

[setting], [scheduler], [parameter] 설정은 [워크플로우 생성] > [설정]을 참고한다.

property#

[Node Description] 작성 중인 노드명 입력

  1. method : 적용할 correlation algorithm 선택 (pearson, spearman)
  2. datasetPath : correlation 을 계산할 csv 파일 경로 선택
  3. savePath : 결과가 저장될 폴더경로 선택
  4. jupyterIP : jupyterIP 입력
  5. delimeter : csv 파일의 구분자 입력
  6. columnHeader : 데이터 컬럼 헤더여부 선택
  7. rowHeader : 데이터 로우 헤더여부 선택
