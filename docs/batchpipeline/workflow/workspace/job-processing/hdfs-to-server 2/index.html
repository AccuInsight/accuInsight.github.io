hdfsToServer

Definition#

HDFS 파일/폴더를 원격 서버에 복사 하는 노드이다. 좌측 [Flow구성]노드 중 [hdfsToServer]노드를 drag & drop 한 후 Property 항목을 입력한다.

Set#

[setting], [scheduler], [parameter] 설정은 [워크플로우 생성] > [설정]을 참고한다.

property#

[Node Description] 작성 중인 노드명 입력

  1. hdfs파일경로(불러오기) : 복사할 파일 경로 선택
  2. dest : 원격서버 정보
  • IP주소 : 원격서버 IP 정보
  • 저장경로(path) : 파일을 저장할 경로
  • PORT : SCP 활용 가능 포트
  1. user : SCP에 활용할 계정
  2. password : SCP에 활용할 password

Example#

kbtest 클러스터 hdfs에 적재된 파일을 외부서버에 전달한다.

  1. [hdfsToServer] 노드 property 구성

  1. 워크플로우 실행 후 파일이 외부서버에 전송되었다.
