#SQL

SQL query를 직접 입력하여 데이터를 처리한다. 좌측 [데이터처리(고급)]노드 중 [SQL]노드를 drag & drop 한다. Property 패널의 [더보기+] 버튼을 누르면 입력가능한 전체 Property 항목을 볼 수 있다.

  1. query : 데이터처리를 위한 select문 입력
  2. overwriteSchema : SQL결과를 데이터 스키마에 저장
