addConditionCol

#addConditionCol

조건식을 입력하여 True/False에 매핑되는 값을 새로운 컬럼에 추가한다. 좌측 [데이터처리(고급)]노드 중 [addConditionCol]노드를 drag & drop 한다.

  1. expression : 사용자가 원하는 조건식을 직접 입력한다(띄어쓰기 필수). expression을 추가할 경우 좌측 [+]버튼을 선택한다.
    • customCondition : 조건식
    • trueValue : 조건식이 True일 경우 매핑 될 값
    • falseValue : 조건식이 False일 경우 매핑 될 값
    • newColumn : 새로 생성할 컬럼명(미입력시 조건식을 적절하게 지환[필드명조건값]하여 지정
