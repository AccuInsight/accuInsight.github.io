mlPipeline

#mlPipeline

ML 모델을 불러와서 예측 결과를 출력한다. 좌측 [데이터처리(고급)]노드 중 [mlPipeline]노드를 drag & drop 한다. Property 패널의 [더보기+] 버튼을 누르면 입력가능한 전체 Property 항목을 볼 수 있다.

  1. modelPath - ML 모델 경로를 지정
  2. useLatestModel : mlTrain 노드(스케줄)의 학습결과 모델 사용 여부. 체크시 mlTrain노드의 modelPath와 동일한 폴더를 선택해야 한다.
  3. overwriteSchema - ML결과로 데이터 스키마 저장
