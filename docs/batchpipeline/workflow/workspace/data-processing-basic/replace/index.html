replace

#replace

특정 컬럼의 값을 지정한 값으로 치환한다. 좌측 [데이터처리(기본)]노드 중 [replace]노드를 drag & drop 한다. Property 패널의 [더보기+] 버튼을 누르면 입력가능한 전체 Property 항목을 볼 수 있다.

  1. to_replace : 변경전 값을 입력한다. 문자열 입력시 쌍따옴표("")를 사용하여 문자열을 감싸줘야 한다.
  2. value : 대치값(변경후 값) 입력한다. 문자열 입력시 쌍따옴표("")를 사용하여 문자열을 감싸줘야 한다.
  3. subset : 목록 중 기준 컬럼을 선택한다. 전체 컬럼을 대상으로 할 경우 생략한다.
