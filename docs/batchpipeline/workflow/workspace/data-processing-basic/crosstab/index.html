#crosstab

열의 빈도 테이블을 계산한다. 좌측 [데이터처리(기본)]노드 중 [crosstab]노드를 drag & drop 한 후 Property 항목을 입력한다.

  1. col1 : 빈도표 작성을 위한 첫 번째 기준컬럼을 선택한다. 결과테이블에서 행에 위치한다.
  2. col2 : 빈도표 작성을 위한 두 번째 기준컬럼을 선택한다. 결과테이블에서 열에 위치한다.

hdfs에 저장된 테이블에서 co11=category, col2=group을 각각 선택할 경우 결과는 아래와 같다.

  • hdfs에 저장된 원테이블

  • crosstab 노드를 활용하여 빈도테이블 계산한 결과 테이블 : category 기준으로 group 별 빈도테이블이 계산됨을 확인할 수 있다.
