dataJoin

#dataJoin

2개의 노드로부터 데이터를 Join한다. 좌측 [데이터처리(기본)]노드 중 [dataJoin]노드를 drag & drop 한 후 Property 항목을 입력한다.

  1. col1 : 첫 번째 노드 기준 컬럼을 선택한다.
  2. col2 : 두 번째 노드 기준 컬럼을 선택한다.
  3. how : join 유형을 선택한다.

선택할 수 있는 join 유형은 다음과 같다.

join유형설명
inner조인 조건에 맞는 행만 검색
outer조인 키에 대응되는 row가 없는 경우 null로 추출
left_outercol1에서 선택한 조인 키에 대응되는 row가 없는 경우 null로 추출
right_outercol2에서 선택한 조인 키에 대응되는 row가 없는 경우 null로 추출
left_semi작성필요
left_anti작성필요
cross두 노드의 카티션 프로덕트(곱집합)결과를 추출
