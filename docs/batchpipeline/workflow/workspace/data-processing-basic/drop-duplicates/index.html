dropDuplicates

#dropDuplicates

컬럼 선택 후 중복값이 있으면 제거한다. 좌측 [데이터처리(기본)]노드 중 [dropDuplicates]노드를 drag & drop 한다.

  1. subset : 중복값 확인을 위한 컬럼을 선택한다. 확인해야 할 컬럼이 여러 개일 경우 좌측 [+]버튼을 선택한다.
    • col : 중복값 확인 대상 컬럼
  • category 컬럼에 중복 값이 있는 행을 찾아 제거한다.

  • category, group에 중복 값이 있는 행을 찾아 제거한다.
