RDB 불러오기

#RDB 불러오기

RDBMS에 저장된 데이터를 Batch Pipeline으로 불러오기 위해 사용하는 노드이다.

좌측 [데이터불러오기]노드 중 [RDB불러오기]노드를 drag & drop 한 후 Property 항목을 입력한다. Property 패널의 [더보기+] 버튼을 누르면 입력가능한 전체 Property 항목을 볼 수 있다.

  1. username : database 사용 계정을 입력한다.
  2. driver : JDBC 드라이버를 입력한다.
  3. password : 패스워드를 입력한다.
  4. url : JDBC 접속을 위한 URL 을 입력한다.
  5. tableName : 접속할 테이블의 정보를 입력한다.
  6. sqlQuery : SQL을 입력한다. (우측 버튼을 이용하여 SQL 편집기를 이용할 수 있다.)
  7. 테이블스키마 : column 및 dataType를 정의한다(우측 버튼을 이용하여 전체 파싱 및 일부 추가 가능)
    • [스키마정보 가져오기] 버튼을 선택하면 컬럼이 자동 설정된다.
    • 자동설정된 정보는 ETL 작업에서 활용할 수 있다.

Property 정보 입력 후 [연결테스트]를 통해 입력정보가 정확한지 확인할 수 있다.

[목록열기] 버튼 선택하여 기 저장한 RDB목록을 적용할 수 있으며, [목록열기]로 팝업된 [JDBC 연결 관리] 창의 [신규생성] 버튼을 활용해서 RDB 접속정보를 입력할 수 있다.
