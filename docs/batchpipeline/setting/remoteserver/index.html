원격서버관리 개요#

원격서버 SSH 접속 정보를 관리한다. 워크플로우 생성/수정 시 Rscript, serverToHdfs 노드에서 원격서버 접속 정보를 로드하여 사용할 수 있다.

① 검색 : 기 등록된 원격서버 정보가 많을 경우 서버정보를 입력하여 검색할 수 있다.

② 추가 : 원격서버 접속 정보를 추가한다.

③ 삭제 : 원격서버 접속 정보를 목록에서 체크 후 삭제 버튼을 클릭하여 삭제한다.

④ 기 등록된 원격서버 목록을 전체 선택시 사용한다. 일괄삭제 등의 작업에 필요하다.

⑤ 편집 : 원격서버 접속 정보를 편집한다. 접속포트, 계정정보의 변경사항 발생시 사용한다.

⑥ 삭제 : 기 등록된 원격서버 중 선택한 정보만 삭제한다.

원격서버 추가/편집#

원격서버 접속 정보의 추가/편집버튼 클릭 시 팝업 화면에서 등록/수정/초기화를 할 수 있다. SSH 접속을 위해 필요한 IP, Port 및 계정정보 등은 시스템 관리자에게 문의한다.

A. Name : 원격서버 접속 정보 구분을 위해 사용하는 이름

B. IP 정보 : 원격서버 IP

C. PORT 정보 : 원격서버 PORT

D. User Name : 원격서버 유저 ID

E. Password : 원격서버 유저 패스워드
