세션관리#

세션관리 개요#

워크플로우 생성/수정 시 데이터 스냅샷, 스키마 조회, ml 컬럼가져오기 기능을 위한 spark livy 세션을 관리한다.

① 검색 필터 : 버튼을 클릭하여 필터창을 활성화 하고 검색할 클러스터명을 입력하여 검색

② 재 검색 : 세션 리스트를 재 조회한다.

③ 클러스터 목록 및 spark livy 세션을 조회한다.

A. ID : 클러스터 ID

B. 클러스터명 : 클러스터 이름

C. 클러스터 타입 : dhp, on-premise의 클러스터 구분

D. 정보 : 클러스터 생성시 입력한 설정 정보 간략 출력 (우측의 … 아이콘 mouse over시 간추린 정보 확인가능)

E. 세션ID : spark livy의 세션 ID

F. 상태 : 세션의 상태 busy(작업중), idle(정상 대기 상태), starting(시작중), stop(정지), unknown(확인불가)

G. 소유자 : 클러스터 생성자

④ 세션 생성 : 세션을 생성한다.

⑤ 세션 삭제 : 세션을 삭제한다.

⑥ 로그 조회 : 세션의 로그를 조회한다.

⑦ 결과 조회 : 세션에서 실행된 spark 코드 및 실행 결과 조회

로그 조회#

spark livy 세션의 로그를 조회한다.

결과 조회#

수행된 spark 코드 및 수행결과를 조회한다. 이전에 실행한 이력도 조회 가능하며 실패 시 에러 로그를 조회 가능하다.
