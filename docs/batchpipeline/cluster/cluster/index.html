cluster

#cluster

사용자가 접근할 수 있는 클러스터를 등록/편집할 수 있고, 현재 등록된 전체 클러스터 목록을 조회할 수 있다.

① ‘검색 필터’를 클릭하여 필터 창을 활성화 하고 검색할 클러스터이름을 입력하여 검색

② Action

종류설명
신규등록클러스터 생성시 ‘신규등록’ 버튼 클릭하여 생성 팝업 활성화 하여 진행
삭제삭제할 클러스터를 목록 에서 체크 후 ‘삭제’버튼 클릭하여 클러스터 삭제
새로고침새로고침 버튼 클릭하여 클러스터 목록 재 조회

③ 현재 로그인한 ID를 통해 생성된 클러스터 목록 출력

종류설명
ID클러스터 ID
클러스터명클러스터 이름
설정클러스터 생성시 입력한 설정 정보 출력(우측 '…' 아이콘 mouse over시 간추린 정보 확인가능)
생성시간클러스터 생성시간
생성자클러스터 생성자
권한버튼다른 사용자에게 클러스터 조회/기록/실행 권한 부여
편집버튼클릭하여 클러스터 설정정보 편집 가능
삭제버튼클릭하여 해당 클러스터 삭제

#클러스터 생성 및 편집

① 등록 및 편집: 신규등록 버튼 또는 편집 버튼 클릭하여 신규클러스터 등록 팝업 활성화 -> 클러스터이름, 설정정보등을 입력한 뒤 저장하여 생성/편집

② 위와 같은 표시는 필수 값이므로 반드시 입력해야 생성 가능
