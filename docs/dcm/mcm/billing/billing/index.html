빌링

사용 현황#

  1. 클라우드 별 비용
    • 조회 기간의 클라우드 별 사용 비용의 비율
  2. 기간 별 클라우드 비용
    • 조회 기간의 클라우드 별 사용 금액, 단위 USD
  3. 계정 별 비용
    • 조회 기간의 계정 별 사용 비용의 비율
  4. 기간 별 계정 비용
    • 조회 기간의 계정 별 사용 금액, 단위 USD
  5. Account
    • 계정의 아이디
  6. 클라우드
    • 계정의 클라우드 정보
  7. 설명
    • 계정의 설명
  8. 등록일
    • 계정의 등록일
  9. 기간
    • 조회 기간
  10. 비용
    • 조회 기간의 사용 금액의 총합, 단위 USD
  11. Action
    • 상세 보기 기능이 있습니다.

  1. 검색 필터
    • 클라우드, 등록일, 기간으로 리스트를 검색할 수 있습니다.
  2. 리플레쉬
    • 사용현황 리스트를 갱신 합니다.
  3. 상세 보기
    • 계정 사용현황을 팝업으로 서비스/사용자로 구분해서 정보를 보여 줍니다.

Account 상세정보#

서비스#

  1. 서비스
    • 서비스 별 상세 내역을 표시 합니다.
  2. 사용자
    • 사용자 별 상세 내역을 표시 합니다.
  3. 서비스 별 비용
    • 조회 기간의 서비스 별 비용의 비율
  4. 기간 별 서비스 비용
    • 조회 기간의 서비스 별 비용, 단위 USD
  5. 서비스
    • 클라우드에서 사용한 서비스 이름
  6. 사용량
    • 조회 기간의 서비스 사용량의 총합 AWS는 사용량의 단위가 없고, GCP는 사용량의 단위는 GB
  7. 비용
    • 조회 기간의 서비스 사용 비용의 총합, 단위 USD

사용자#

  1. 사용자 별 비용
    • 조회 기간의 사용자 별 비용의 비율
  2. 기간 별 사용자 비용
    • 조회 기간의 사용자 별 비용, 단위 USD
  3. 사용자
    • 계정의 사용 권한이 있는 사용자 아이디
  4. 비용
    • 조회 기간의 사용자 사용 비용의 총합, 단위 USD
