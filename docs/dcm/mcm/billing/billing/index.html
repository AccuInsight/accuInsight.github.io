빌링

#사용 현황

① 메뉴진입

전체 리스트가 검색 조건없이 로드되어 화면에 표시 됩니다.

② 필터 조회 영역

검색필터 버튼을 클릭하면 입력창이 활성화 됩니다. 등록일과 기간을 가지고 검색합니다.

③ 차트 조회 영역

사용자별 기간별 조회된 내용을 차트로 표시한다. 검색한 기간이 30일 이상을 검색하면 월별 30일 보다 적으면 일별로 차트를 표시한다.

④ 상세정보

검색할 계정의 Action 컬럼에서 상세 정보아이콘을 클릭하면 상세 정보팝업이 나타납니다. 상세정보에는 서비스 탭과 사용자 탭이 있습니다.

⑤ 상세정보 - 서비스

상세 정보 팝업을 띄우면 먼저 보이는 탭입니다. 선택한 계정이 사용한 각 서비스 별 사용량과 비용을 차트와 리스트로 보여줍니다.

⑥ 상세정보 - 사용자

상세 정보 팝업에서 사용자 탭을 선택하면 나타납니다. 선택한 계정에 등록된 사용자별 비용을 차트와 리스트로 보여줍니다.
