클러스터 템플릿 관리

#AWS 클러스터 템플릿 관리

① 헤더영역

사용가능한 Accuinsight+ 서비스와 Multi Cloud Manager에서 서비스하는 메뉴가 화면에 표시 됩니다.

② 메뉴진입

전체 리스트가 검색 조건 없이 로드되어 화면에 표시 됩니다.

③ 템플릿 생성 버튼

우측상단 템플릿 생성 버튼을 클릭하면 탬플릿 생성 팝업이 활성화 됩니다.

필수 입력사항

  • 닉네임
  • Master
  • Core
  • Task
  • 최대노드수 (최소 3)

Master, Core, Task 입력창에 자동완성 기능이 포함되어 있습니다.

필수 입력사항 입력 후 생성버튼을 클릭하면 템플릿이 생성됩니다.

④ 필터조회 영역 검색필터 버튼을 클릭하면 입력창이 활성화 됩니다.

닉네임, 생성자로 조회 가능합니다. 입력된 문자가 포함된 데이터만 로드되어 리스트에 표시됩니다.

⑤ 템플릿 수정 버튼

수정할 템플릿의 Action 아이콘 중 수정 아이콘을 클릭하면 수정 팝업이 활성화 됩니다.

필수 입력사항과 입력 방법은 템플릿 생성과 동일합니다.

변경사항 입력 후 수정 버튼을 클릭하면 템플릿이 수정됩니다.

⑥ 템플릿 삭제

삭제할 템플릿들을 체크 후 우측상단 삭제버튼을 클릭하면 삭제확인 팝업이 나타나며 일괄 삭제 됩니다.

삭제할 템플릿의 Action아이콘 중 삭제 아이콘을 클릭하면 삭제확인 팝업이 나타나고 확인 후 삭제가 가능합니다.
