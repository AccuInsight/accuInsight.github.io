#계정 관리

① 메뉴진입

전체 리스트가 검색 조건 없이 로드되어 화면에 표시 됩니다.

Note

계정관리 AWS 의 IAM 권한을 통해 Role 방식으로 권한이 주어져 있어 계정등록은 따로 하지 않아도 됩니다.

② 사용자 권한 관리

수정할 계정의 권한 컬럼에서 권한 관리 아이콘을 클릭하면 권한 관리 팝업이 나타납니다. 아래 권한 컬럼에서 권한을 변경하며 없음, 조회, 조회/생성 3가지 권한이 있습니다. 권한을 변경 후 적용 버튼을 누르면 수정된 권한으로 변경됩니다.

Note

계정 수정, 삭제 버튼 Role 방식의 계정관리는 계정의 수정, 삭제를 할 수 없습니다.
