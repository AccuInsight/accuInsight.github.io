모델 모니터링 관리#

Data drift 기반의 분석모델 성능 모니터링 기능을 제공합니다.

  1. 상단에서 프로젝트 선택
  2. 관리 메뉴에서 모델 모니터링 선택
  3. 테이블 목록

삭제 : 성능 모니터링 리소스 삭제 및 관련 정보 삭제

모델 모니터링 생성#

  1. 모델 모니터링 생성
    • 우측 상단의 ‘+ 신규 생성’ 버튼을 눌러 모니터링 생성
