권한

사용자 관리#

  1. 요약
    • 계정ID 별 정보 리스트가 화면에 나타납니다.
  2. 필터 조회 영역
    • 검색필터 버튼을 클릭하면 입력창이 활성화 됩니다. 등록일과 기간을 가지고 검색합니다.
  3. 참여 프로젝트
    • 셀의 숫자를 클릭하게 되면 해당 계정(ID)의 참여프로젝트 목록이 나타납니다.

계정권한 수정#

  1. 계정 권한 수정
    • 계정권한 수정을 클릭하면, 관리자 여부와 계정 활성 상태를 수정 가능 하도록 셀이 활성화 됩니다. 수정이 완료 된 후에 수정완료를 클릭하면, 셀이 비활성화 되고 수정이 완료됩니다.
  2. GitLab 패스워드 초기화
    • 해당계정의 깃Lab 패스워드를 초기화 합니다.
