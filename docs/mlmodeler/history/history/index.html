Training History

#Training History

학습한 이력 및 결과를 조회한다.

#학습 이력 조회

학습한 이력들의 정보를 조회한다.

  • 상단의 Pipeline 명 또는 Dataset 명, 날짜로 검색한다.
  • 좌측 목록에서 학습한 이력들의 정보(파이프라인 명, 상태, 데이터셋, 에러 메시지, eval, 시작시간, 종료시간)를 확인한다.
  • 하나를 선택 하면 우측에서 Pipeline 정보(데이터셋 및 알고리즘)와 학습 Property(프로퍼티) 값을 확인한다.

#학습결과 조회

하나를 선택하여 우측 상단의 Results (결과 조회) 버튼을 클릭하여 학습 결과를 조회한다.

#Training Result조회

Training Result 탭을 클릭하여 학습 결과(평가)를 조회한다.

  • 마지막 Item이 머신러닝 알고리즘일 경우, 테스트셋에 대한 Evaluation 결과가 보여진다.
  • 알고리즘에 따라 보여지는 그래프가 다르다.

#Model Description 조회

Model Description 탭을 클릭하여 학습 결과(모델)를 조회한다.

  • 마지막 Item이 머신러닝 알고리즘일 경우, Model description 클릭 시 학습된 모델 결과가 보여진다.

#Spark ML code조회

Spark ML code 탭을 클릭하여 Spark에서 실행한 Code를 조회한다.

#학습이력 Filter

좌측 3번째 란에 Filter에서 타입에 따라 학습이력 조회가 가능하다.

  • Workspace Training : ML Modeler의 workspace 화면에서 학습을 수행한 것
  • Scheduled Training : Batch Pipeline의 workflow에서 ML 학습을 수행한 것
  • Training History
    • 학습 이력 조회
    • 학습결과 조회
    • 학습이력 Filter
