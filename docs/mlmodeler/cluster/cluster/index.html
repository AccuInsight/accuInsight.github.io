Cluster

Cluster#

학습, 예측 시 사용할 클러스터 목록을 관리한다.

클러스터 관리#

클러스터 조회#

현재 로그인한 ID를 통해 생성되거나 권한 있는 클러스터 목록을 조회한다. 검색 필터를 통해서 조회한다.

  • 상단의 Pipeline 명 또는 Dataset 명, 날짜로 검색한다.
  • 좌측 목록에서 학습한 이력들의 정보(파이프라인 명, 상태, 데이터셋, 에러 메시지, eval, 시작시간, 종료시간)를 확인한다.
  • 하나를 선택 하면 우측에서 Pipeline 정보(데이터셋 및 알고리즘)와 학습 Property(프로퍼티) 값을 확인한다.

클러스터 등록#

우측 상단 '신규 등록' 버튼을 클릭하여 등록할 클러스터 정보를 입력한다.

클러스터 수정#

기능 컬럼에 '편집' 아이콘을 클릭하여 편집 창에서 클러스터 정보를 수정한다. ( DHP에서 만들어진 클러스터나 클러스터 생성자가 아닌 경우는 수정 불가능/조회만 가능)

클러스터 삭제#

우측 상단 '삭제' 버튼 또는 기능 컬럼에 '삭제' 아이콘을 클릭하여 클러스터를 삭제한다. ( DHP에서 만들어진 클러스터나 클러스터 생성자가 아닌 경우는 삭제 불가능)

클러스터 권한 관리#

권한 컬럼에 '권한' 아이콘을 클릭하여 해당 클러스터를 사용할 사용자에게 권한을 부여한다.

Livy Session 관리#

Livy Session 관리#

SparkML 작업을 실행하기 위한 Livy Session을 관리한다. 클러스터 관리 화면에서 사용할 클러스터의 설정 컬럼에 연필 아이콘을 클릭한다.

Livy Session 조회#

Livy Session들이 조회되고, 현재 상태들이 보여진다. “IDLE”이 되면 학습 시킬 준비가 되어있는 상태이다. (“Busy”는 학습 중인 상태이다.)

상태의 상세 내용은 아래와 같다.

Livy Session 등록#

우측 상단 New 버튼을 클릭하여 Livy Session을 등록한다.

입력할 정보는 다음과 같다.

Livy Session Log 조회#

Logs 컬럼의 돋보기 아이콘을 클릭하여 Session의 로그를 조회한다. Session이 만들어지지 않는 경우에 원인을 파악할 수 있다.

Livy Session Statements 조회#

Action 컬럼의 리스트 아이콘을 클릭하여 Statement 상태와 진행률, Code, Output을 조회한다.

Livy Session 삭제#

Action 컬럼의 휴지통 아이콘을 클릭하여 Session을 삭제한다.
