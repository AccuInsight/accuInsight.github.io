Dataset

Dataset#

Dataset 조회#

등록된 Dataset을 조회 및 관리한다.

  • Dataset 이름이나 기간을 이용하여 Dataset을 검색한다.
  • New(신규등록) 버튼을 클릭하여 등록한다.
  • Dataset Name(데이터셋명)을 클릭하여 Dataset Overview 창을 통해 데이터를 확인한다.
  • 다운로드 아이콘을 클릭하여 해당 Dataset을 로컬로 다운로드한다.
  • 연필 아이콘을 클릭하여 해당 Dataset을 수정한다.
  • 삭제 아이콘을 클릭하거나 선택 후 상단의 삭제버튼을 클릭하여 Dataset을 삭제한다.
  • 적용 아이콘을 클릭하여 해당 Dataset을 Workspace(워크스페이스) 화면을 열어 학습 데이터로 적용한다.

Dataset Overview#

Dataset Name을 클릭하여 등록된 Dataset정보를 조회한다.

  • Data Overview : 데이터가 테이블로 표현된다.
  • 통계정보보기: 컬럼을 선택하고 버튼을 클릭하면 해당 컬럼의 통계정보를 표시한다.
    • NumericType 경우: count, mean, min, max, stddev 표시
    • StringType 경우: distinct count 표시

Dataset 등록#

New(신규등록) 버튼을 클릭하여 등록한다.

  • Data의 위치를 선택한다.
    • HDFS/S3/ICOS에 존재하는 경우, HDFS/S3/ICOS 상의 경로를 입력한다. 우측Browse버튼을 클릭하여 선택하거나 텍스트로 입력한다.
    • 로컬에 존재하는 경우, 로컬상의 경로를 입력한다. 우측Browse버튼을 클릭하여 로컬 경로에서 선택한다.
    • HIVE에 존재하는 경우, /데이터베이스명/테이블명/partitionOption정보를 입력한다. 우측Browse버튼을 클릭하여 입력하거나 위의 형태로 직접 입력한다.
  • Dataset Name: 사용자 지정 (다른 데이터셋과 중복되지 않는 이름이어야 함)
  • Format : 데이터 포맷 (default:Csv(text))
  • Header 유무 : 데이터에 헤더 정보가 포함되었는지 여부
  • Delimiter : 데이터의 구분자 (default: “,”)
  • Description : 데이터 설명

Dataset 수정#

등록된 Dataset정보를 수정한다.

  • 이름과 설명, 컬럼의 Type을 수정할 수 있다.
  • Save버튼을 클릭하여 수정내용을 저장한다.
  • Close버튼을 클릭하여 수정내용을 취소하고 화면을 닫는다.
  • Init 버튼을 클릭하여 수정내용을 초기화한다.

Dataset 삭제/다운로드/적용#

  • [삭제] 삭제 아이콘을 클릭하거나 선택 후 상단의 삭제버튼을 클릭하여 Dataset을 삭제한다.
  • [다운로드] Action 란에 다운로드 아이콘을 클릭하여 다운로드 받는다.
  • [적용] Action 란에 적용 아이콘을 클릭하여 해당 Dataset을 Workspace 화면을 열어 학습 데이터로 적용한다.
