Tutorial

개요#

ML Modeler를 활용하여 파이프라인 디자인, 실행, 학습모델 생성 및 예측하는 작업을 수행한다.

튜토리얼 수행을 위한 사전 준비사항은 다음과 같다.

사전 준비사항#

클러스터#

AccuInsight+ ML Modeler 사용을 위해 AccuInsight+ DHP 서비스를 사용해야 한다.

  1. DHP 서비스링크(https://accuinsight.cloudz.co.kr/dhp) 클릭
  2. [클러스터] > [클러스터 생성하기] 활용하여 튜토리얼 수행을 위한 클러스터 생성 default에서 아래 값 변경
  • 데이터 노드 : 3
  • yarn container's mem size (G) : 10G

DHP 사용법은 https://accuinsight.cloudz.co.kr/#/manual/dhp 참고

데이터 준비#

kaggle에서 제공하는 데이터를 준비한다.

  • 학습 데이터 다운로드 : https://www.kaggle.com/cono94/hr-comma-separatedcsv
  • kaggle 계정이 없을 시 (우클릭 다른이름으로 링크 저장) : https://raw.githubusercontent.com/sagara00/datasets/master/HR_retention.csv

데이터 등록#

dataset 탭 클릭하여 다운로드한 데이터셋을 등록한다.

list에서 생성된 dataset 클릭. data sample 보기, 컬럼 선택 후 통계정보 보기 확인

ML 파이프라인 디자인#

등록한 데이터셋 로드하여 전처리 작업을 수행한다.

  • binarizer

binarizer

  • nomalizer

  • one-hot-encoder
  • ML 알고리즘 input feature 정의 (전처리한 feature를 입력하고, label이 될 left는 제외)

  • output 입력(사용자 정의)하고 label을 left로 설정

  • pipeline 저장

  • pipeline tab 확인. 선택 후 open

ML 파이프라인 실행 및 학습모델 저장#

run 버튼 클릭해서 ML pipeline을 실행한다.

  • 학습 모델 내용 입력

  • 하둡 클러스터 선택

  • Now Running(혹은 idle) 버튼 클릭하면 Running 상태 조회 창에서 상태 조회

  • 결과보기를 클릭해서 학습결과 확인

예측#

  • model 탭에서 저장 결과 확인 및 예측

  • dataset 선택 후 예측 실행

  • 예측 결과 dataset 저장

  • dataset 탭에서 예측결과 확인
