AccuInsight+




AccuInsight

AccuInsight+는 모든 산업 도메인 영역에 적용되는 Common Platform으로서
고객사의 Data-Driven 비즈니스 전환을 가능케 하는 Data Science 기능을 제공함









기능요소

Data Science 영역의 니즈와 플랫폼 관점의 문제를 해결하기 위한 데이터 & AI 영역의
개발 / 분석 / 운영의 Total Out-Sourcing 가능
Data Science Needs
Platform Needs
  1) Data Scientist 독립된 개발 환경 제공  4) Public / Private 클라우드 환경 선택 용이
  2) Data 분석 모델 자산화 및 지속 갱신 가능  5) 검증된 오픈 소스 기술 활용된 플랫폼 제공
  3) 선정된 모델 운영 시스템으로 전환 속도 빠름  6) 고객사 환경下 파이프 라인 커스터 마이징









차별화 Point

AccuInsight+는 사용자 별 독립된 분석 환경을 제공하며 고객사에 최적화된 솔루션을
인프라 환경에 구애 없이 설치하여 사용 가능함

Advanced Workbench

  • 사용자 별 독립된 분석 환경

  • 분석 Asset 공유 / 협업

  • 대용량 병렬처리 분석 지원
Customization

  • 데이터 파이프라인 최적화 지원

  • 솔루션 성능 최적화

  • 레거시 연동 커스터 마이징
Any Infrastructure

  • Private / Public Cloud 지원
    (AWS, GCP, AZURE)

  • On-Prem. 인프라 설치 가능







Platform Architecture

AccuInsight+는 Data 전처리로 부터 분석과 운영의 Science 전 영역을
한번에 지원하는 Multi-Cloud 기반 DevOps 머신 러닝 플랫폼







Pipeline

AccuInsight+ Pipeline : Drag & Drop UI 기반 데이터 전처리 / 머신 러닝
AccuInsight+ Modeler : 직접 코딩 가능한 Advanced 분석가 대상 모델링 / 운영 배포

AccuInsight+ Pipeline (초급 분석가 대상)
AccuInsight+ Modeler (고급 분석가 / ML 엔지니어)
  • Spark을 활용한 인터렉티브 전처리
  • Oozie 기반의 워크플로우 관리
  • 대용량 데이터 분산 머신러닝 (SparkML)
  • 코딩없이 ML Pipeline 구성/실행
  • 50개 전처리 / 37개 머신러닝 알고리즘

  • Advanced 개발 환경 제공
    (Jupyter, Rstudio)

  • DL/ML code model 형상관리 손쉽고
    빠르게 웹 형태의 API 제공 가능

  • Rest API 형태의 모델 배포

  • Swagger UI를 통한 단위 테스트 지원

  • Model 재 배포를 위한 모니터링/시각화
